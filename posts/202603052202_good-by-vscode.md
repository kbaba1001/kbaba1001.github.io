VS Code をやめてみることにした。理由は

- Linux ユーザーなのでなんとなく Microsoft 依存が嫌
  - (そんな事言ったら github もやめないとだめだが...)
- AI 関係とかで余分な機能が増えてきたように感じる
- 気分転換

最初は [kate](https://kate-editor.org/) を試したけど、ちょっと慣れなかったので、結局 [VS Codium](https://vscodium.com/) にした。
本当は [Zed](https://zed.dev/) にしたかったのだが、これは Linux で日本語入力 (fcitx) がうまくできなかった...。

[VS Codium](https://vscodium.com/) は自由なライセンスにしたバージョンの VS Code で、基本的には VS Code と変わらない。

## 結局ターミナルを便利にしたほうがいい

昔からエディタは GUI ツールが好きなのだが、それ以外はターミナルで動かすツールが好きだ。
（vim, emacsでもいいんだけども...）

最近よく使っている CUI ツールは次:

- [lazygit](https://github.com/jesseduffield/lazygit)
- [tmux](https://github.com/tmux/tmux)
- Claude Code
- Github Copilot Cli

なんかもう少し面白みがほしいね...。
